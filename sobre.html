Sobre o Projeto

O GITHUB é uma plataforma de hospedagem, usada para facilitar o trabalho de desenvolvedores. A ideia é desenvolver uma plataforma parecida com o YouTube, mas com um foco mais educativo e colaborativo. O código do site está sendo hospedado no GitHub, o que facilita o trabalho em equipe e o controle de versões. Cada integrante pode adicionar melhorias, corrigir erros e testar novas funções. Assim, o projeto cresce com a contribuição de todos os desenvolvedores envolvidos.

O layout do site foi pensado para ser moderno e intuitivo, o usuário pode criar uma conta, enviar videos e interagir com outros perfis. Tudo isso está sendo desenvolvidos com o HTML, CSS e JavaScript.
